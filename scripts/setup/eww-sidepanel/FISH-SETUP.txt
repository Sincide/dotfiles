EWW SIDEPANEL - FISH SHELL SETUP
=================================

DOWNLOAD THESE FILES:
====================
✅ setup.fish              # Main setup script
✅ eww.yuck                # EWW configuration (Fish-compatible)
✅ eww.scss                # Styling (same as bash version)
✅ toggle_sidebar.fish     # Toggle script
✅ system.fish             # System monitoring script
✅ music.fish              # Music control script  
✅ weather.fish            # Weather information script
✅ network.fish            # Network status script

SETUP STEPS:
===========

1. Create download folder:
   mkdir ~/Downloads/eww-fish
   cd ~/Downloads/eww-fish

2. Download all files above to this folder

3. Create scripts directory:
   mkdir scripts

4. Move script files:
   mv system.fish scripts/
   mv music.fish scripts/
   mv weather.fish scripts/
   mv network.fish scripts/

5. Your folder should look like:
   eww-fish/
   ├── setup.fish
   ├── eww.yuck
   ├── eww.scss
   ├── toggle_sidebar.fish
   └── scripts/
       ├── system.fish
       ├── music.fish
       ├── weather.fish
       └── network.fish

6. Run setup:
   chmod +x setup.fish
   ./setup.fish

MANUAL SETUP (if setup.fish fails):
===================================

mkdir -p ~/.config/eww/scripts

cp eww.yuck ~/.config/eww/
cp eww.scss ~/.config/eww/
cp toggle_sidebar.fish ~/.config/eww/
cp scripts/*.fish ~/.config/eww/scripts/

chmod +x ~/.config/eww/toggle_sidebar.fish
chmod +x ~/.config/eww/scripts/*.fish

echo 'bind = SUPER, grave, exec, ~/.config/eww/toggle_sidebar.fish' >> ~/.config/hypr/hyprland.conf

USAGE:
======
Super + grave (`) to toggle sidebar
Or run: ~/.config/eww/toggle_sidebar.fish

DEPENDENCIES:
============
eww playerctl brightnessctl bc jq curl

Arch: sudo pacman -S eww playerctl brightnessctl bc jq curl
Ubuntu: sudo apt install eww playerctl brightnessctl bc jq curl

CUSTOMIZATION:
=============
• Change weather city in scripts/weather.fish (line 8)
• Modify colors in eww.scss
• Edit widgets in eww.yuck

TROUBLESHOOTING:
===============
• Test scripts: fish ~/.config/eww/scripts/system.fish cpu
• Check EWW: eww logs
• Start daemon: eww daemon &
• Reload config: eww reload
